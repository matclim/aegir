# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-04-08

### Features

- Pythia8 and fixed-target event generation with PythiaParallel support
- Geant4 detector simulation via direct worker integration
- RNTuple output with validation histograms
- Pluggable geometry: builtin, GDML, and GeoModel providers
- Multi-threaded workflows with configurable concurrency
